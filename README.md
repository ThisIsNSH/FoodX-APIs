# FoodX-APIs
APIs, Personal Hotel Dashboard &amp; Online Add Hotel &amp; Item Form for Food Ordering App FOODX

## What APIs are made?

* POST /status
* GET /status
* POST /order
* GET /hotel
* GET /hotel/menu
* GET /hotel/order
* POST /hotel

## What views are made?

* Add Hotel & Menu
* View Orders & Post order status

## What technologies are used?

* NodeJS
* MongoDB
* Express
* EJS
* Body-Parser
* HTML/CSS
* etc
